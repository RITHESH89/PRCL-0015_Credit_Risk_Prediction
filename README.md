# 💳 Credit Score Prediction – PRCL-0015

## 📌 Project Overview
This project focuses on predicting the **creditworthiness of existing credit card customers** for GoodCredit Bank.  
The objective is to identify customers likely to default and reduce financial risk using Machine Learning.

The model classifies customers into **Good Credit** or **Bad Credit** categories based on historical behavior and financial attributes.

---

## 🏦 Business Problem
Banks face significant losses due to credit defaults. Early identification of risky customers allows better credit decisions and portfolio risk management.

GoodCredit Bank aims to build a predictive model that:

- Predicts customer default risk
- Improves credit approval strategy
- Reduces Non-Performing Assets (NPAs)
- Enhances credit scoring process



## 🎯 Target Variable

**Bad_label**

| Value | Description |
|------|-------------|
| 0 | Customer has Good Credit History |
| 1 | Customer has Bad Credit History (30+ Days Past Due) |

---

## 📊 Project Goal
Develop a machine learning model that outperforms the benchmark performance:

**Benchmark Gini Score:** `37.9`

Success Criteria:
- Achieve higher predictive accuracy
- Improve Gini coefficient
- Maintain model interpretability
