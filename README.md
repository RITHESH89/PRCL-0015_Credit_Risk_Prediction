# Credit Score Prediction

A Machine Learning project that predicts customer credit score categories (**Good, Standard, Poor**) using financial and behavioral data.


##  Overview
This project uses a classification model to analyze customer financial information and predict creditworthiness.  
It helps financial institutions automate loan risk assessment.


##  Model
- Random Forest Classifier
- Supervised Machine Learning (Classification)


## Tech Stack
- Python
- Pandas & NumPy
- Scikit-learn
- Matplotlib & Seaborn


## Workflow
Data Preprocessing → Feature Engineering → Model Training → Evaluation → Prediction


## Result
- Model Accuracy: **~85%**
- Predicts: Good / Standard / Poor credit score
